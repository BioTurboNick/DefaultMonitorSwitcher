# DefaultMonitorSwitcher

Automatically reverts the Windows primary display from an HDTV back to a desktop
monitor when HDTV activity ceases, and switches it forward when activity migrates
to the HDTV. Switches the default audio playback device alongside the display.
Runs as a system tray application with no main window.

---

> **Warning:** This project was vibecoded with [Claude Code](https://claude.com/claude-code)
> and is tailored to one specific hardware setup (two desktop monitors + one HDTV).
> Monitor device paths, audio endpoint names, and several hardcoded assumptions
> reflect that environment directly. **Do not use this software on your own system
> without thoroughly reviewing the code and configuration for your own setup.**
